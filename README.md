# PyGameOfLife

An interactive OOP Game of Life written in Python 3 and Pygame.

Controls:

    Right Mouse....Set Dead
    Left Mouse.....Set Alive
    
    Esc............Exit
    R..............Reset
    C..............Clear
    P..............Pause
    O..............Output
    G..............Grid
    I..............Info
    
    1..............Spawn Selected
    2..............Decrease Selection Index
    3..............Increase Selection Index
